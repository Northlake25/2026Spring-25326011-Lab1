# 严重问题修复总结

## ✅ 已修复的四个严重问题

### 1. 批量添加只添加第一句

**问题**：ID生成太快导致重复，只添加了第一句

**修复**：
```cpp
qint64 baseTime = QDateTime::currentMSecsSinceEpoch();
for (int i = 0; i < couplets.size(); ++i) {
    QString id = QString("q%1_%2").arg(baseTime).arg(i);  // 基础时间+索引
    ...
}
```

**结果**：可以正确添加所有联

### 2. 标点符号不算字

**问题**：添加时没有去除标点，导致字数计算错误

**修复**：
```cpp
// 去除标点符号，只保留汉字
QString upperClean, lowerClean;
for (const QChar& ch : sentences[0].trimmed()) {
    if (ch.script() == QChar::Script_Han) {
        upperClean += ch;
    }
}
```

**结果**：字数计算正确

### 3. 题库不标注上下句

**问题**：题库列表显示"(上句)"或"(下句)"标注

**修复**：
```cpp
QString display = QString("[%1] %2")
    .arg(q.id)
    .arg(q.content);  // 删除位置标注
```

**结果**：题库列表简洁清晰

### 4. 诗词库删除ID字段

**问题**：诗词库需要手动输入ID

**修复**：
- 删除ID输入框
- 自动生成ID
