# 足球比赛记录器 (Football Match Recorder)

一个基于 HarmonyOS 开发的足球比赛赛后记录工具,用于记录已结束的比赛进程。

## 项目简介

这是一个专门为赛后记录设计的工具。适用于比赛结束后整理比赛数据、生成比赛报告等场景。

## 功能特性

### 核心功能

- **比赛管理**
  - 创建新比赛
  - 查看比赛列表
  - 查看比赛详情
  - 删除比赛记录

- **球队信息**
  - 主队/客队名称
  - 比赛场地
  - 比赛日期和时间
  - 赛事名称
  - 裁判信息

- **比赛事件记录**
  - 进球事件(含助攻)
  - 点球进球(标记P)
  - 乌龙球
  - 罚丢点球
  - 黄牌/红牌
  - 换人事件

- **比分计算**
  - 自动计算常规时间比分
  - 点球大战独立计分
  - 乌龙球自动计入对方

- **时间轴显示**
  - 上半场/下半场分隔
  - 加时赛分隔
  - 点球大战分隔
  - 时间轴居中,左右分栏显示

### 高级功能

- **自定义比赛时长**
  - 上半场时长(默认45分钟)
  - 下半场时长(默认45分钟)
  - 加时赛时长(默认30分钟)

- **点球大战支持**
  - 自动序号分配(P1, P2, P3...)
  - 点球比分独立显示
  - 不影响常规比分

- **数据持久化**
  - 本地数据存储
  - 自动保存

## 技术栈

- **开发框架**: HarmonyOS
- **开发语言**: ArkTS (TypeScript-based)
- **UI框架**: ArkUI (声明式UI)
- **数据存储**: Preferences API
- **文件系统**: fs API

## 使用说明

### 创建比赛

1. 点击"开始记录比赛"
2. 填写比赛基本信息:
   - 主队名称（必填）
   - 客队名称（必填）
   - 比赛场地
   - 比赛日期和时间
   - 赛事名称
   - 裁判
3. 设置比赛选项:
   - 加时赛(可自定义时长)
   - 点球大战
4. 点击"创建比赛"

### 记录事件

1. 在比赛详情页点击"记录事件"
2. 选择事件类型，从左至右的选项依次是（在这个版本中，文字暂时无法显示）:
   - 进球
   - 换人
   - 黄牌
   - 红牌
   - 罚丢点球
   - 乌龙球
3. 填写事件信息:
   - 比赛时间(如: 45', 90', P（P即点球大战阶段）)
   - 球员信息
   - 其他相关信息
4. 点击确认保存

### 时间输入说明

- **常规时间**: 直接输入分钟数,如 `45`, `90`
- **加时赛**: 输入超过常规时间的分钟数,如 `105`, `120`
- **点球大战**: 输入 `P`,系统自动分配序号(P1, P2, P3...)

### 比分显示

**主界面卡片:**
- 无点球大战: `电子与通信工程学院 1-2 理学院`
- 有点球大战: `中山大学理学院(6) 1-1 (5)中山大学医学院`

**比赛详情页:**
- 常规比分显示在上方
- 点球比分显示在下方: `点球 5 : 4`

## 数据模型

### 比赛信息 (FootballMatch)

```typescript
interface FootballMatch {
  id: string;                    // 比赛ID
  homeTeam: Team;                // 主队
  awayTeam: Team;                // 客队
  venue: string;                 // 比赛场地
  date: string;                  // 比赛日期
  time: string;                  // 比赛时间
  competition: string;           // 赛事名称
  referee: string;               // 裁判
  homeScore: number;             // 主队得分
  awayScore: number;             // 客队得分
  homePenaltyScore?: number;     // 点球大战主队得分
  awayPenaltyScore?: number;     // 点球大战客队得分
  status: string;                // 比赛状态
  events: MatchEvent[];          // 比赛事件列表
  config: MatchConfig;           // 比赛配置
}
```

### 比赛事件类型

- `GOAL` - 进球
- `OWN_GOAL` - 乌龙球
- `PENALTY_MISSED` - 罚丢点球
- `YELLOW_CARD` - 黄牌
- `RED_CARD` - 红牌
- `SUBSTITUTION` - 换人

## 开发说明

### ArkTS 限制

- 不支持 `Omit`/`Partial` 类型
- 不支持展开运算符 `...`
- 不支持 `any`/`unknown` 类型
- 必须显式声明接口
- 静态方法不能使用 `this` 关键字

### UI 组件

- 使用 `@State` 装饰器管理状态
- 使用 `@Builder` 装饰器定义组件
- 使用 `Column`/`Row` 布局
- 使用 `List` + `ForEach` 渲染列表
