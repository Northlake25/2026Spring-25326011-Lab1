# 实时显示功能实现

## ✅ 已完成的功能

### 1. 实时显示输入

**功能描述**：
- 在输入框输入时，实时显示在网格的当前行
- 类似Wordle的实时反馈体验
- 自动去除标点符号显示

**实现方式**：
- 连接`QLineEdit::textChanged`信号
- 调用`updateCurrentRowDisplay()`方法
- 实时更新网格显示

### 2. 灰色字提示标记

**功能描述**：
- 已确认不存在的字用灰色背景标记
- 在输入时就能看到哪些字已经排除
- 帮助玩家避免重复输入无效字

**实现方式**：
- 获取`getGrayCharacters()`集合
- 检查每个输入字符是否在灰色集合中
- 应用特殊样式`getGrayCharStyle()`

### 3. 智能显示

**功能特点**：
- 自动标准化输入（去除标点）
- 根据题目类型显示正确字数
- 空位显示空白方框
- 实时更新，无需等待提交

## 🔧 技术实现

### 实时显示流程

```
用户输入 → textChanged信号 → onInputTextChanged()
    ↓
updateCurrentRowDisplay()
    ↓
标准化输入（去除标点）
    ↓
清除当前行显示
    ↓
重新显示每个字符
    ↓
检查是否为灰色字 → 应用对应样式
```

### 样式定义

**普通字符**：
```css
font-size: 18px;
font-weight: bold;
border: 2px solid #D3D6DA;
border-radius: 5px;
```

**灰色字**：
```css
font-size: 18px;
font-weight: bold;
background-color: #D3D6DA;
color: #787C7E;
border-radius: 5px;
```

**空白方框**：
```css
font-size: 18px;
border: 2px solid #D3D6DA;
border-radius: 5px;
```
